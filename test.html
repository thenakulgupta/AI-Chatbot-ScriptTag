AI Chatbot Integration Example

Basic Integration

To integrate the AI Chatbot into your website, simply add this script tag to your HTML:

<script src="https://ai-chat-script-projects.stackmind.tech/ai-chatbot.js"></script>

Advanced Configuration

You can customize the chatbot behavior using data attributes:

<script src="https://ai-chat-script-projects.stackmind.tech/ai-chatbot.js" data-default-message="Welcome! How can I help you?" data-category="support" data-auto-open="true"></script>

Chat Categorization

Organize your chats by category to better manage different types of conversations:

<!-- Support page --> <script src="ai-chatbot.js" data-category="support"></script> <!-- Sales page --> <script src="ai-chatbot.js" data-category="sales"></script> <!-- General page --> <script src="ai-chatbot.js" data-category="general"></script>

This allows you to:

  • Separate support tickets from sales inquiries
  • Track conversation types and analytics
  • Route different categories to appropriate teams
  • Maintain organized chat history by purpose

JavaScript API

You can also control the chatbot programmatically:

// Open the chatbot AIChatbot.open(); // Close the chatbot AIChatbot.close(); // Toggle the chatbot AIChatbot.toggle(); // Send a message programmatically AIChatbot.sendMessage("Hello, how are you?"); // Get current session ID const sessionId = AIChatbot.getSessionId();

API Integration

The chatbot expects your API to accept POST requests with this format:

{ "message": "User's message", "history": [ { "user": "Previous user message", "bot": "Previous bot response", "timestamp": "2024-01-01T12:00:00.000Z" } ], "timestamp": "2024-01-01T12:00:00.000Z", "userAgent": "Browser user agent", "url": "Current page URL" }

And return responses in this format:

{ "message": "Bot's response message", "status": "success" }
